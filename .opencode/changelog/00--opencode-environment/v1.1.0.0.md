## [**1.1.0.0**] - 2026-01-29

This release updates the OpenCode environment with work on cognitive memory system (spec 079), bug remediation (spec 080), agent system improvements (spec 005). The goal is to make the platform clearer, more reliable, and easier to use day to day.

### Cognitive Memory System (Spec 079)
- Need: This part of the release improves how the environment saves, finds, and uses context.
- What changed: Memory Decay Scoring Power-law Decay: R(t,S) = (1 + 0.235 × t/S)^(-0.5) replaces exponential decay. Prediction Error Gating: Prevents duplicates (≥0.95), handles contradictions (0.90-0.94), links related (0.70-0.89). 5-state Memory Model: HOT/WARM/COLD/DORMANT/ARCHIVED with configurable thresholds.
- Why it matters: That matters because saved context is easier to find, easier to trust, and less likely to get in the way.
