## [**1.0.6.0**] - 2026-01-21

### 1. CORE + ADDENDUM V2.0 Template Architecture
- Need: This part of the release makes agent behavior more consistent across the supported runtimes.
- What changed: Compositional Template Model: Monolithic templates replaced with modular core + addendum architecture. Core Templates (322 Lines Of Content): spec-core.md, plan-core.md, tasks-core.md, impl-summary-core.md. Addendum Folders: level2-verify (184 lines of content), level3-arch (220 lines of content), level3plus-govern (190 lines of content).

- Restart Required - Restart OpenCode to load updated templates and command definitions.

- Compose Script - Use./compose.sh --all to regenerate levels if customizing core templates.
- No Breaking Changes - All existing spec folders and APIs maintain backward compatibility.
- Optional Migration - Copy verbose templates to existing specs: cp templates/verbose/level_2/* specs/###-feature/.
