## [**1.0.0.0**] - 2025-12-29

First official release - Complete OpenCode Dev Environment with Spec Kit, Skills Framework, Semantic Memory, Gate Enforcement, and 11 automation scripts integrated into Claude Code. This release focuses on making the environment easier to use and easier to trust.

### New Features
- Need: This part of the release introduces the main capabilities added in this version.
- What changed: Spec Kit: Unified documentation system with automation, slash commands, integrated semantic memory, and sub-folder versioning. Skills Framework: 8 domain-specific skills that auto-load based on task. Semantic Memory: Custom tool connection layer server with hybrid search (vector + full-text search), 6 importance tiers, and proactive context surfacing.
- Why it matters: That matters because the environment can do more useful work without extra setup.

### Memory System
- Need: This part of the release improves how the environment saves, finds, and uses context.
- What changed: Persistent memory across sessions, models, projects. Gate 3 enforcement ensures spec folders on every change. ADRs in decision-record.md, searchable forever.
- Why it matters: That matters because saved context is easier to find, easier to trust, and less likely to get in the way.
