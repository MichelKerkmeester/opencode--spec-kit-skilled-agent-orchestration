This release updates the OpenCode environment with work on new specialized agents, enhanced agents, enterprise orchestration patterns. The goal is to make the platform clearer, more reliable, and easier to use day to day.

### 1. New Specialized Agents

- What changed: @research - Technical Investigation With Evidence Gathering → /spec_kit: research (Steps 3-7). @speckit - Spec Folder Documentation For Level 1-3+ → /spec_kit: plan (Step 3). @review - Code Review With Pattern Validation (READ-ONLY) → /spec_kit: implement (Step 11).
