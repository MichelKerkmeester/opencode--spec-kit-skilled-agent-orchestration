## [**2.1.3.0**] - 2026-02-18

This release updates the OpenCode environment with work on features. The goal is to make the platform clearer, more reliable, and easier to use day to day.

### Features
- Need: This part of the release introduces the main capabilities added in this version.
- What changed: Copilot Fleet Upgrade: context, handover, review, speckit, write all moved to github-copilot/claude-sonnet-4-6. Parent Model Inheritance: research and debug agents no longer hard-pin opus - they now inherit whichever model dispatched them, enabling flexible model assignment without file edits. Claude Code Agents: context and handover promoted from haiku → sonnet; review promoted from opus → sonnet.
- Why it matters: That matters because the environment can do more useful work without extra setup.
