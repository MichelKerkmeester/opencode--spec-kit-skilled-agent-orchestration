## [**1.0.1.2**] - 2025-12-29

Project-Local Indexing - Adds per-project Narsil index support for isolated codebase indexing and team collaboration. This release focuses on making the environment easier to use and easier to trust.

### Local Index Management
- Need: This part of the release improves how the environment saves, finds, and uses context.
- What changed: Project-local indexing -.narsil-index/ directory instead of shared ~/.cache/narsil-mcp/. Persistence Control: -persist flag for index persistence across sessions. Custom Paths: -index-path option for index location flexibility.
- Why it matters: That matters because saved context is easier to find, easier to trust, and less likely to get in the way.
