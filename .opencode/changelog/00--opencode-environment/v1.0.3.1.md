## [**1.0.3.1**] - 2026-01-05

Agent Configuration Refactor - Updates agent configuration to OpenCode v1.1.1+ format. This release focuses on making the environment easier to use and easier to trust.

### Configuration Updates
- Need: This part of the release makes agent behavior more consistent across the supported runtimes.
- What changed: Agent Frontmatter - Tools: object → permission: object (v1.1.1+ format). Agent Files - Write.md · Orchestrate.md: Consolidated tool permissions into unified permission block. Agent Template: Updated with v1.1.1 format, granular permissions example, deprecation note.
- Why it matters: That matters because the same instructions now behave more predictably in different runtimes.

## Upgrade

- Existing agents with tools: format continue to work (backwards compatible). New agents should use the permission: format. See agent_template.md for migration examples.
