## [**1.0.0.7**] - 2025-12-29

Semantic Search Upgrade - Major performance enhancement with Qwen3 embeddings and progressive indexing for large projects. This release focuses on making the environment easier to use and easier to trust.

### Qwen3 Embeddings
- Need: This part of the release improves qwen3 embeddings.
- What changed: Qwen3 Embeddings: mlx-community/Qwen3-Embedding-0.6B-4bit-DWQ with 4-bit quantization.
- Why it matters: That matters because it makes the environment easier to use and maintain.

### Progressive Indexing
- Need: This part of the release improves how the environment saves, finds, and uses context.
- What changed: Progressive Indexing: Incremental scope indexing for large projects.
- Why it matters: That matters because saved context is easier to find, easier to trust, and less likely to get in the way.

### Architecture
- Need: This part of the release makes agent behavior more consistent across the supported runtimes.
- What changed: AGENTS.md now frontend/backend agnostic for broader compatibility.
- Why it matters: That matters because the same instructions now behave more predictably in different runtimes.
