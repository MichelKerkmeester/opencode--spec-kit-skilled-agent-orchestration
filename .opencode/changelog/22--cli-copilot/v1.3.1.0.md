## [**1.3.1.0**] - 2026-03-14

This update warns Copilot not to call itself through this skill. It prevents circular delegation and points people to the built-in Copilot features they already have. The goal is to make the workflow easier to follow and easier to trust.

---

### Stop circular delegation

**Problem:** A Copilot session could try to delegate back into Copilot through this same skill, which adds no value and creates confusion.

**Fix:** The main guide now states clearly that Copilot should use its native features instead of calling itself through this wrapper. The workflow is simpler and avoids unnecessary loops.
