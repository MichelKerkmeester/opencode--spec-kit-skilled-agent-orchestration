## [**1.1.0.0**] - 2026-03-07

This update refreshes the model guide so it matches the current Copilot tool and adds clearer notes about model selection and hands-off mode. It helps operators choose the right setup without trial and error.

---

### Up-to-date model options

- Need: The documented model list no longer matched what Copilot actually offered.
- Change: The model tables were expanded and corrected so they reflect the current choices.
- Why it matters: People can pick a model with more confidence and less guesswork.

---

### Clearer non-interactive use

- Need: Users also needed better guidance for selecting a model and reducing follow-up questions during automated runs.
- Change: The docs now explain direct model selection and the flag that suppresses user prompts.
- Why it matters: Copilot is easier to use in automated or lower-touch workflows.
