## [**1.0.0.0**] - 2026-03-02

**Initial release enabling any AI assistant to orchestrate GitHub's Copilot CLI for cross-AI tasks.**

---

## Highlights

### New

- **Cross-AI Orchestration** — Any AI assistant as conductor, Copilot CLI as executor for supplementary tasks (multi-model generation, cloud delegation, collaborative planning, autopilot execution)
- **7-Intent Smart Routing** — PLANNING, CLOUD_DELEGATE, GENERATION, REVIEW, AGENT_DELEGATION, TEMPLATES, PATTERNS with canonical `score_intents()` / `select_intents()` pattern
- **Multi-Provider Model Support** — 7+ models from 3 providers: Claude Opus/Sonnet/Haiku (Anthropic), GPT-5.3-Codex/GPT-5 mini/GPT-4.1 (OpenAI), Gemini 3 Pro (Google)
- **Cloud Delegation** — `/delegate` or `&prompt` prefix pushes tasks to GitHub's cloud coding agent for background execution
- **Plan Mode** — Collaborative planning before implementation with Shift+Tab toggle
- **Autopilot** — Fully autonomous multi-step execution without approval prompts
- **Repository Memory** — Persistent codebase convention indexing across sessions
- **Agent System** — Built-in Explore (read-only) and Task (read-write) agents + custom Markdown-based agent profiles
- **Skill Advisor Integration** — INTENT_BOOSTERS, MULTI_SKILL_BOOSTERS, PHRASE_INTENT_BOOSTERS entries in `skill_advisor.py`
- **Documentation (7 files)** — SKILL.md (8 sk-doc sections), README.md, `references/cli_reference.md`, `references/integration_patterns.md`, `references/copilot_tools.md`, `references/agent_delegation.md`, `assets/prompt_templates.md`
- **4-Way Comparison Table** — Copilot CLI vs Claude Code vs Gemini CLI vs Codex CLI across 20 capability dimensions in `copilot_tools.md`
- **Implementation via Gemini CLI** — Skill files generated by Gemini CLI from Level 2 spec docs, reviewed and aligned by Claude
