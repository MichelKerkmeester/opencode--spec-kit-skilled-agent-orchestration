## Highlights

### Updated

- **18-Model Support** — Expanded model table from 7 to 18 models matching current `copilot --help` output: added Claude Opus 4.6 Fast, Claude Opus 4.5, Claude Sonnet 4.5, Claude Sonnet 4, GPT-5.4, GPT-5.2-Codex, GPT-5.2, GPT-5.1-Codex-Max, GPT-5.1-Codex, GPT-5.1, GPT-5.1-Codex-Mini; renamed Gemini 3 Pro to Gemini 3 Pro Preview

- **`--no-ask-user` Flag** — Documented autonomous mode flag that suppresses interactive questions
- **Open Issue Reference** — Linked [github/copilot-cli#1048](https://github.com/github/copilot-cli/issues/1048) for the missing `--reasoning-effort` CLI flag
