## [**1.1.0.0**] - 2026-03-08

This release explains the correct way to set Codex reasoning depth and removes guesswork around the available options. It helps operators avoid using the wrong command pattern.

---

### Clearer reasoning-depth setup

- Need: People were trying to use the wrong command style for reasoning depth.
- Change: The docs now list the supported levels and explain the correct configuration-based setup path.
- Why it matters: Automated runs are less likely to fail because of an avoidable syntax mistake.

---

### Consistent guidance across the package

- Need: The flag and model guidance needed to say the same thing everywhere.
- Change: The main guide, reference docs, comparison notes, and templates were all updated together.
- Why it matters: Users can move between documents without getting mixed messages.
