## [**1.3.3.0**] - 2026-03-28

This update requires the calling AI to pass along the active spec folder when it delegates work to Codex. It avoids a common stop point where the delegated agent would otherwise pause for missing project context.

### Pass project context forward

- Need: Delegated Codex runs could stop early because they did not know which spec folder had already been approved.
- Change: The delegation rules now require the calling AI to include that spec-folder context in the handoff when it exists.
- Why it matters: Non-interactive delegation is more likely to keep moving instead of pausing for avoidable questions.
