## [**1.3.1.0**] - 2026-03-14

This update warns Codex not to call itself through this skill. It prevents circular delegation and points people to the built-in Codex features they already have.

---

### Stop circular delegation

- Need: A Codex session could try to delegate back into Codex through this same skill, which adds no value and creates confusion.
- Change: The main guide now states clearly that Codex should use its native features instead of calling itself through this wrapper.
- Why it matters: The workflow is simpler and avoids unnecessary loops.
