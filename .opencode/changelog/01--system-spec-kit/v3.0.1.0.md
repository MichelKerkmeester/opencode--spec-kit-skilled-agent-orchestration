## Highlights

### P0 Blockers Fixed (5)

- **Scope-aware lineage keys** -- `buildLogicalKey()` now includes governance scope tuple (tenant, user, agent, session, shared_space) via SHA-256 hash prefix, preventing cross-scope collisions on `memory_lineage` and `active_memory_projection`
- **Embedding dimension guard** -- `get_embedding_dim()` replaced ad hoc env checks with canonical provider resolution via `getStartupEmbeddingDimension()`; fail-fast guard rejects dimension mismatches before DB bootstrap
- **Checkpoint merge restore atomicity** -- Merge-mode restore passes normalized scope into pre-clears; savepoint-per-table rollback prevents partial state; `partialFailure`/`rolledBackTables` metadata surfaces failures
- **Shared-memory caller authentication** -- `validateCallerAuth()` requires caller identity on all admin mutations; new-space creation is admin-only; membership changes require owner or admin role
- **Reconsolidation merge reachability** -- `executeMerge()` now upserts `active_memory_projection`, records lineage, syncs BM25; bridge returns `newMemoryId` instead of archived predecessor ID

### Sprint 1: Search + Data Integrity (15 fixes)

- **Hybrid search threshold units** -- Fallback constants changed from fractional (0.3) to percentage (30) to match vector search contract
- **Channel preservation in fallback** -- Caller-disabled channels (`useBm25: false`, etc.) now respected through all fallback tiers
- **Degree channel gated by useGraph** -- `useGraph: false` now removes both `graph` and `degree` from active channels
- **Unified lexical weighting** -- Adaptive fusion uses one consistent model regardless of graph/degree channel presence
- **Confidence truncation reordered** -- Moved after MMR, co-activation, and folder scoring so later stages can rescue candidates
- **Token budget accuracy** -- Estimates include nested `traceMetadata`/`sources` fields; oversized first results handled
- **PE scope filtering** -- Scoped saves now require exact equality; NULL scope fields treated as mismatch
- **Atomic save file-first** -- File promotion before DB indexing; compensating cleanup on rename failure
- **Transaction scope narrowed** -- `BEGIN IMMEDIATE` acquired only for mutation phase, not across async reconsolidation
- **Chunking quality gate** -- Quality gate + reconsolidation run before chunking fork
- **Projection writes mandatory** -- `active_memory_projection` upsert part of required mutation contract
- **SQLiteVectorStore DB pinning** -- Each store instance pins its own DB handle through all operations
- **BM25 document shape centralized** -- `buildBm25DocumentText()` used by all mutation paths (save, merge, lineage, chunking, update, unarchive)
- **BM25 assistive auto-merge sync** -- Archived rows evicted from BM25 singleton immediately
- **BM25 reconsolidation sync** -- Merge path uses canonical document shape for add/remove

### Sprint 2: Correctness + Schema (13 fixes)

- **Causal graph diamond traversal** -- Recursion-path tracking replaces global visited set; alternate paths preserved
- **Direction-aware drift response** -- `memory_drift_why` returns separate `incoming`/`outgoing` buckets with per-edge direction
- **Cumulative strength propagation** -- Child strength = clamp(parentStrength * edgeStrength * weight)
- **Migration fail-hard** -- DDL failures re-throw for required changes; `schema_version` gated on compatibility check
- **History backfill ordering** -- v23 repairs `memory_history.spec_folder`; migrations run before history init
- **Constitutional-tier enforcement** -- Real table-rebuild migration; fail startup on legacy constraints
- **v12 audit preservation** -- Rename-copy-drop migration maps `similarity_score`->`similarity`, `notes`->`reason`
- **Dimension-aware embedding cache** -- Cache key includes `(content_hash, model_id, dimensions)`; stale vectors rejected
- **Provider fallback metadata** -- `requestedProvider`, `effectiveProvider`, `fallbackReason`, `dimensionChanged` tracked
- **Startup API key validation** -- `resolveStartupEmbeddingConfig()` couples validation to first cloud-provider resolution
- **Anchor-mode text preservation** -- Non-anchor regions preserved as first-class chunks
- **AST-aware structure fallback** -- Fenced code blocks stay atomic; `# comments` inside code no longer split
- **Atomic chunk child writes** -- Insert + metadata in one transaction with cleanup on partial failure

### Sprint 3: Security + Governance (9 fixes)

- **Status tool authentication** -- `shared_memory_status` derives scope from authenticated caller, not request body
- **Domain-specific error codes** -- Handler errors use appropriate codes (CAUSAL_GRAPH_ERROR, TRAVERSAL_ERROR, etc.)
- **Error sanitization** -- Internal exception text stripped before returning to MCP clients
- **Response envelope completeness** -- Profiles preserve metadata fields; server exceptions always wrapped in envelope
- **DB rebind propagation** -- `db-state` listener broadcasts to all registered consumers on handle swap
- **Access-tracker reset** -- Accumulators cleared on DB connection change
- **Scoped continuation cursors** -- Progressive-disclosure tokens carry scope metadata; validated on resume

### Sprint 4: Remaining P1s (22 fixes)

- **Lineage identity check** -- PE SUPERSEDE only chains same-path files; cross-path recorded as causal link
- **Parser hardening** -- YAML body override blocked, malformed anchor nesting rejected, checklist state preserved
- **Context handler** -- Intent-aware ranking in focused mode, context-based resume detection, proper error responses
- **Trigger quality** -- Stopword filter for common-word phrases, structured error on frontmatter extraction failure
- **Dedup improvements** -- Force-save skips dedup rejection, chunked saves run content-hash dedup
- **Session learning** -- `task_postflight` scoped by session_id, multiple completion entries per task_id
- **Trigger scope wiring** -- `memory_match_triggers` scope fields connected to matching logic
- **Graph signal accuracy** -- Self-loops counted once, degree cache invalidation on all mutation surfaces, 0.15 cap enforced
- **Index scan robustness** -- Content-hash secondary detection, replacement-first deletion
- **Feature flag enforcement** -- `SPECKIT_GRAPH_REFRESH_MODE=off` actually disables enrichment
- **Eval framework** -- Dashboard preserves per-channel breakdown, ablation enforces ground-truth alignment guard
