## [**3.0.0.0**] - 2026-03-27

**022 Hybrid RAG Fusion: 19-phase program completion** -- Delivers the full `022-hybrid-rag-fusion` program as the release that turns `system-spec-kit` into a platform-scale documentation and memory system rather than a loose collection of scripts and MCP handlers. The shipped surface now spans **33 MCP tools across 7 layers with a 7600-token budget**, **14 commands**, **255 documented features across 21 categories**, **25 SQLite tables**, **26+ feature flags**, **20 validation rules**, **22 scripts**, and **81 CORE+ADDENDUM v2.2 templates**.

This release hardens the memory engine around a **5-channel hybrid RAG stack**: Vector Voyage `1024d` semantic retrieval with **OpenAI** and **HuggingFace** provider support, plus **FTS5**, **BM25**, **causal graph traversal** and **typed degree scoring** with a fan-effect cap of `0.15`. Retrieval is now organized as a **4-stage pipeline**: **Gather** (parallel candidate generation plus constitutional injection), **Score+Fuse** (RRF `1/(K + rank)` with intent-tuned `K` in `{10, 20, 40, 60, 80, 100, 120}` plus 8 post-fusion signals), **Rerank** (local GGUF reranking via `node-llama-cpp` with MPAB chunk collapse), and **Filter** (score immutability, state gating, confidence labeling, and rollout-aware filtering).

The fused-scoring path is materially richer than earlier releases. Stage 2 combines co-activation at `+0.25 / sqrt(n)`, FSRS retrievability `R(t,S) = (1 + (19/81) * t/S)^-0.5` from the same research lineage validated on `100M+` Anki reviews, an interference penalty of `-0.08` for `Jaccard > 0.75`, a cold-start lift of `0.15 * exp(-h/12)` capped at `0.95`, session-recency boost capped at `0.20`, causal 2-hop propagation, 7 intent profiles, and a channel minimum-representation floor of `0.005`. Around that core, the release lands **3-tier fallback**, **Simple/Moderate/Complex complexity routing**, **6 importance tiers** with constitutional memories at **3.0x / never-decay**, **5 lifecycle states** from `HOT` through `ARCHIVED`, **6 causal relation types**, **prediction-error save gating**, query intelligence with **HyDE**, decomposition and context-pressure thresholds at **60% / 80%**, session attention decay at `0.85^distance`, roughly **50% deduplication savings**, about **93% ANCHOR retrieval savings**, deny-by-default shared memory, and an evaluation harness covering **12 metrics** and **110 ground-truth queries**.
