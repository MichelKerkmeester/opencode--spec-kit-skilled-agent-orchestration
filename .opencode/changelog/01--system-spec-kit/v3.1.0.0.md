## Phase 10: Fix All 15 Deferred P2 Findings

### Graph & Traversal (Agent 1 - 3 fixes)

1. **Snapshot-consistent traversal** (P2-005) -- `memory_drift_why` reads now wrapped in a single DB transaction for consistent edge+node snapshots during concurrent link/unlink
2. **Truncation flag on high-degree traversal** (P2-006) -- Edge queries return `truncated: boolean` + `limit: number` when MAX_EDGES_LIMIT hit; propagated through causal chain response
3. **Degree cache invalidation on DB reinit** (P2-027) -- `db-state.ts` rebind listener clears graph-search degree cache; prevents stale scores after checkpoint restore

### Lineage & Shared Memory (Agent 2 - 4 fixes)

4. **Conflict-aware lineage retry** (P2-010) -- Version assignment retries 3x on UNIQUE constraint violation, re-reading max version per logical key
5. **Graph walk rollout docs fixed** (P2-011) -- Removed unsupported `full` state from environment variable docs; kept only `off`, `trace_only`, `bounded_runtime`
6. **Shared-memory defaults corrected** (P2-012) -- Documentation now correctly shows default as `false`, matching runtime
7. **Shared-memory enablement auth** (P2-015) -- `handleSharedMemoryEnable()` now requires admin caller identity via `validateCallerAuth()`

### Intent & Reconsolidation (Agent 3 - 3 fixes)

8. **Ranked intent classification** (P2-019) -- Classifier returns `rankedIntents` array (top-3) alongside backward-compatible `intent` field for multi-facet queries
9. **BM25 reconsolidation repair** (P2-020) -- Failed BM25 updates trigger immediate repair attempt; failure surfaces as structured warning instead of silent log
10. **Session state ordering** (P2-033) -- Auto-discovered spec folders included in first `saveSessionState()` call for resume recovery

### Trigger Matcher (Agent 4 - 2 fixes)

11. **Unicode-aware tokenization** (P2-034) -- `\p{L}` Unicode property class for word boundaries; unified normalization via `normalizeUnicode()`; CJK fallback for non-delimited scripts
12. **Indexed trigger scan** (P2-035) -- Token/n-gram candidate index narrows regex scan from O(all_triggers) to O(matching_candidates); verified at 500+ triggers

### Embedding & Architecture (Agent 5 - 3 fixes)

13. **Embedding timeout handling** (P2-038) -- `withTimeout()` wired into HF-local inference; `Promise.race` warmup deadline for OpenAI/Voyage cloud providers
14. **Envelope contract cleanup** (P2-040) -- `autoSurfacedContext` moved from top-level sibling into `meta.autoSurfacedContext` inside standardized envelope
15. **Lazy handler imports** (P2-041) -- Handler barrel uses cached lazy loaders instead of eager imports; module graph no longer triggers all singletons at startup
