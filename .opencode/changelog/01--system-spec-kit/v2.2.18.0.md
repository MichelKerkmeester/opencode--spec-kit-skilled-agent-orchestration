### Post-upgrade auto-populate workflow

**Problem:** This release updates Post-upgrade auto-populate workflow.

**Fix:** After upgrade level runs, the AI agent reads existing spec files, identifies [placeholder] patterns in newly injected sections, and replaces them with context-derived content. This matters because the release is clearer and easier to work with.

**Problem:** This release updates Check placeholders.
