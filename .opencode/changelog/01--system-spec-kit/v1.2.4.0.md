## [**1.2.4.0**] - 2026-02-03

This release generate context script fixes  resolving API naming mismatch and template warning suppression. It focuses on making the instructions and release notes easier for non-developers to follow.

---

## What Changed

---

### API naming mismatch

- Need: There was a reliability problem around API naming mismatch.
- Change: Added snake_case export aliases to vector index for backward compatibility with retry manager: initialize_db, get_db, get_memory, get_db_path; fixes vector_index.get_db is not a function error during retry processing.
- Why it matters: This matters because it reduces confusing failures and makes the system more dependable.

---

### Template warning suppression

- Need: People needed clearer guidance about Template warning suppression.
- Change: Added OPTIONAL_PLACEHOLDERS set to template renderer to suppress warnings for V2.2 spec'd-but-unimplemented placeholders: Session Integrity Checks (8), Memory Classification (6), Session Deduplication (3), Postflight Learning Delta (9).
- Why it matters: This matters because the release notes and instructions are easier to read and trust.
