This release post-update MCP recovery guide hardening -- Rewrote the Spec Kit Memory install guide to workflows-documentation standards and added operator-ready recovery and verification guidance for update-related MCP failures. It focuses on making the instructions and release notes easier for non-developers to follow. The goal is to make the workflow easier to follow and easier to trust.
