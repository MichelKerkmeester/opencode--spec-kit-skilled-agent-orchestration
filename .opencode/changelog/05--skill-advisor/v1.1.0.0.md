## Highlights

### New

- **`skill_advisor_runtime.py` runtime layer** -- Added cached skill discovery with mtime invalidation, fast frontmatter-only parsing, and precomputed normalized metadata.
- **`skill_advisor_regression.py` quality harness** -- Added permanent routing regression suite with gate checks and machine-readable report output.
- **`skill_advisor_bench.py` performance harness** -- Added latency and throughput benchmark suite for subprocess one-shot, in-process warm, and batch modes.
- **`fixtures/skill_advisor_regression_cases.jsonl`** -- Added versioned regression dataset for stable routing verification.
- **Structural batch mode** -- Added `--batch-file` and `--batch-stdin` to reduce repeated subprocess overhead.

### Changed

- **Default threshold behavior hardened** -- `--threshold` now keeps dual-threshold safety active by default (`confidence` + `uncertainty`), instead of implicitly switching to confidence-only.
- **Explicit confidence-only override** -- Added `--confidence-only` as an intentional opt-in for confidence-only filtering.
- **Command bridge separation** -- Command bridges are now surfaced as `kind: "command"` and deprioritized unless explicit slash intent is detected (for example `/spec_kit:` and `/memory:save`).
- **Ranking calibration improvements** -- Added intent normalization plus margin-aware and ambiguity-aware confidence calibration for more stable recommendation quality.
- **Health diagnostics expansion** -- `--health` now reports `command_bridges_found` and runtime cache status.

### Performance Snapshot

- **Warm in-process p95** -- `0.6081 ms`
- **Subprocess one-shot p95** -- `46.9855 ms`
- **Batch throughput multiplier** -- `25.8538x` versus one-shot baseline

### Documentation and Governance

- **Script docs synchronized** -- Updated `.opencode/skill/scripts/README.md` and `.opencode/skill/scripts/SET-UP_GUIDE.md` to match new CLI behavior and harness workflows.
- **Level 3 spec completion** -- Completed and validated `.opencode/specs/03--commands-and-skills/advisor-001-refinement` with final validator status: 0 errors, 0 warnings.
