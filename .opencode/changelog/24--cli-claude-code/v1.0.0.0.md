## [**1.0.0.0**] - 2026-03-02

This first release lets outside AI assistants delegate focused work to Claude Code. It packages model guidance, structured handoff patterns, and reference docs so teams can use Claude Code as a deliberate helper instead of an ad hoc fallback.

### Structured delegation into Claude Code

- Need: Other assistants needed a clearer way to decide when Claude Code was the right helper for a task.
- Change: The skill introduced routing patterns for deep reasoning, code editing, review, structured output, and agent-based delegation.
- Why it matters: Requests can be handed off with a better fit and clearer expectations.

### Play to Claude Code's strengths

- Need: Teams needed a simple explanation of what Claude Code is especially good at.
- Change: The package highlights the model tiers, deeper thinking mode, structured output, budget controls, and edit workflow.
- Why it matters: Users can choose Claude Code for the jobs where it adds the most value.

### Reusable support material

- Need: People needed guides and templates they could reuse instead of building prompts from scratch.
- Change: The release shipped a main guide, references, comparisons, and prompt templates.
- Why it matters: Claude Code becomes easier to adopt and easier to use consistently.
