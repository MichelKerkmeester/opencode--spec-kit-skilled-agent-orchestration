# mcp-clickup v1.0.0

## Summary

Created `mcp-clickup` skill — a hybrid CLI+MCP orchestrator for ClickUp project management. Provides intelligent routing between CLI (`cu`, 30+ commands) for daily task operations and MCP (Code Mode, 46 tools) for enterprise features.

## Changes

### Added
- **SKILL.md**: 8-section AI-facing entry point with CLI vs MCP decision matrix, smart routing pseudocode, and ALWAYS/NEVER/ESCALATE rules
- **README.md**: User-facing documentation with trigger_phrases frontmatter, quick start, features, configuration, examples, troubleshooting, and FAQ
- **INSTALL_GUIDE.md**: Dual installation guide for CLI (npm/Homebrew) and MCP (Code Mode), with PATH troubleshooting for system `cu` conflict
- **references/tool_reference.md**: Complete reference for 46 MCP tools across 18 categories with invocation patterns
- **references/cli_reference.md**: Reference for 30+ CLI commands with flags, output modes, and error handling
- **references/workflows.md**: Combined CLI+MCP workflow patterns including daily standup, task lifecycle, cross-tool workflows, and decision guide
- **assets/tool_categories.md**: Priority matrix (8 HIGH, 19 MEDIUM, 19 LOW) with decision flowchart
- **Skill routing**: Updated `skill_advisor.py` to route "clickup" to `mcp-clickup` instead of `mcp-code-mode`
- **Skills catalog**: Updated README.md count (15 -> 16) and added catalog entry

## Pattern Source

Follows hybrid CLI+MCP pattern from `mcp-chrome-devtools` skill, with MCP-only patterns from `mcp-figma`.

## File Count

7 new files in skill directory, 2 files modified for registration.
