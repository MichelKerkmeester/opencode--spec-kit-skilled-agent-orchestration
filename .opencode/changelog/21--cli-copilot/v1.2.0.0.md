## [**1.2.0.0**] - 2026-03-09

This release explains the real non-interactive way to set reasoning depth and removes older workaround guidance. It also narrows the model list to a smaller set of practical recommendations.

### 1. A direct setup path for reasoning depth

- Need: The earlier guidance depended on a workaround that was harder to automate cleanly.
- Change: The docs now explain the direct configuration setting that controls reasoning depth without interactive setup.
- Why it matters: Automated runs are easier to configure and easier to repeat.

### 2. Simpler model guidance

- Need: A very long model catalog made it harder to decide what to use.
- Change: The recommended model list was trimmed to a smaller set of practical choices.
- Why it matters: New users can pick a sensible starting point more quickly.
