## [**1.0.1.0**] - 2026-01-05

This update packages the runner source code inside the skill folder and improves setup guidance. It makes the skill easier to move, share, and configure correctly.

### More portable package

- Need: Users needed the skill and its core source code to travel together as one unit.
- Change: The runner source was embedded directly inside the skill folder.
- Why it matters: The package is more self-contained and easier to reuse in other environments.

### Clearer environment setup

- Need: People were missing the required naming pattern for environment settings.
- Change: The release added a template and stronger install guidance around the correct naming format.
- Why it matters: Setup is less error-prone and troubleshooting is easier.
