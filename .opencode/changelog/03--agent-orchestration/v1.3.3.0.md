This release turns the Claude agent directory from placeholder stubs into full working agent definitions. It also adds a tool-call budget to the orchestrator so large dispatches stay more controlled and easier to reason about.

---

### Full `.claude/agents/` rollout

**Problem:** This part of the agent coordination guidance needed clearer guidance and less room for confusion.

**Fix:** Replaced the one-line stubs with eight complete Claude-native agent definitions. The new files now carry full workflow metadata, tool declarations, and runtime settings instead of acting as empty placeholders.

---

### Orchestrator budget update

**Problem:** People needed clearer support for this new part of the workflow.

**Fix:** Added the Tool Call Budget system to the orchestrator, removed redundant sections, consolidated the capability map, and cut older tables that were repeating the same information. The result is a shorter guide with clearer dispatch rules.
