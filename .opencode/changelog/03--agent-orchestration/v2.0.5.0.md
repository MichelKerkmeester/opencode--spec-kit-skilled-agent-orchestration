This release reorganizes the orchestration guides around the real decision loop operators follow in practice. The result is a shorter, clearer structure, a more visible Nesting Depth Protocol, and better parity across the OpenCode, Copilot, and Claude variants.

---

#### New Section Layout

People needed clearer support for this new part of the workflow. Rebuilt the guide around the actual decision loop: Core Workflow, Agent Routing and Nesting, Task Decomposition and Dispatch, Mandatory Rules, Output Review and Quality, Failure Handling, Synthesis and Delivery, Budget Constraints, Anti-Patterns, and Reference. That makes the guide much easier to scan from top to bottom.

---

#### NDP Moved from Buried §26 to Prominent §2

This part of the agent coordination guidance needed clearer guidance and less room for confusion. Promoted the three-tier agent model (ORCHESTRATOR, DISPATCHER, LEAF) with a hard maximum depth of 3. That makes the nesting rules visible before people start dispatching work.
