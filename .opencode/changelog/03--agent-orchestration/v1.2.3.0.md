## [**1.2.3.0**] - 2026-02-06

Added comprehensive Context Window Budget (CWB) system to orchestrator agent, enabling intelligent result collection across parallel agent dispatches. The main changes here are cwb system and result collection patterns. The goal is to make the workflow easier to follow and easier to trust.

---

### CWB System

**Problem:** This part of the agent coordination guidance needed clearer guidance and less room for confusion.

**Fix:** Budget formula (~150K available tokens), scale thresholds (1-4 direct, 5-9 summary-only, 10-several files-based), pre-dispatch checklist, enforcement points. This makes the agent coordination guidance easier to follow.

---

### Result Collection Patterns

**Problem:** This part of the agent coordination guidance needed clearer guidance and less room for confusion.

**Fix:** Pattern A (Direct), Pattern B (Summary-only), Pattern C (File-based+waves). This makes the agent coordination guidance easier to follow.

---

### The Related File

**Problem:** People needed clearer support for this new part of the workflow.

**Fix:** Added Step 5 (CWB CHECK) to core workflow, new §27-§28 sections. CWB ceiling on parallel-first principle: 10+ agents dispatch in waves of 5. Extended task decomposition with Output Size and Write To fields. This makes the workflow easier to discover and use.
