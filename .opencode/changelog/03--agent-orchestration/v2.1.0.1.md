## [**2.1.0.1**] - 2026-02-25

Ultra-think stabilization release with orchestrator parity across all runtime folders. Caps Ultra-think strategy fan-out to a maximum of 3, aligns depth semantics with single-hop NDP expectations, and updates orchestrator routing in OpenCode, ChatGPT, Claude, and Gemini to include @ultra-think consistently. Codex adapters now reference main OpenCode orchestrator/ultra-think playbooks. The main changes here are strategy cap reduced from max 5 to max 3 and depth expected behavior clarified. The goal is to make the workflow easier to follow and easier to trust.
