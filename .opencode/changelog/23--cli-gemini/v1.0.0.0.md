## [**1.0.0.0**] - 2026-02-28

This first release lets other AI assistants delegate focused work to Gemini CLI. It packages research-friendly routing, live web information access, and reusable guidance so teams can use Gemini more intentionally.

### 1. Structured delegation into Gemini

- Need: Other assistants needed a clear way to decide when Gemini was the right helper for a task.
- Change: The skill introduced routing patterns for generation, review, research, architecture work, and agent-based delegation.
- Why it matters: Requests can be handed off with a better fit and clearer expectations.

### 2. Better support for live information

- Need: Some tasks needed current web information and broader investigation support.
- Change: The package highlights Gemini's live search and investigation strengths as first-class reasons to delegate work there.
- Why it matters: Users can choose Gemini when fresh information matters most.

### 3. Reusable support material

- Need: People needed guides and templates they could reuse instead of building prompts from scratch.
- Change: The release shipped a main guide, references, and prompt templates for common delegation cases.
- Why it matters: Gemini becomes easier to adopt and easier to use consistently.
