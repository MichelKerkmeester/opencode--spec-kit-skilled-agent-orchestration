## [**1.2.0.0**] - 2026-03-04

**Refocused sk-prompt-improver to text-only by removing all visual/creative mode references, aligning the Smart Router with the canonical template, and moving interactive mode orchestration to command files.**

---

## Highlights

### Removed

- **`references/format_guides.md`** — Deleted consolidated format guide (1,123 lines). Individual asset files (`assets/format_guide_*.md`) remain as the canonical format references.
- **`references/interactive_mode.md`** — Deleted interactive mode reference (531 lines). Essential conversation orchestration content (response templates, error recovery, smart defaults, formatting rules, max interactions) moved to command files.
- **Visual/creative frameworks** — Removed VIBE, VIBE-MP, FRAME, MOTION frameworks and EVOKE/VISUAL scoring from `patterns_evaluation.md`. Framework count corrected from 11 to 7.
- **Creative energy level** — Removed Creative energy level routing, round config, perspectives, and mode-specific scoring from `depth_framework.md`.

### Added

- **4 framework deep dives** in `patterns_evaluation.md` — RACE Urgency Patterns, CIDI Process Documentation Patterns, CRISPE Strategic Exploration Patterns, CRAFT Comprehensive Planning Patterns. Each follows the YAML template + pattern summary table format of existing deep dives.
- **CLEAR per-dimension scoring rubrics** in `patterns_evaluation.md` — 0-3/4-6/7-8/9-10 rubric tables for all 5 dimensions (C, L, E, A, R) with dimension floor thresholds quick-reference table.
- **Step 1a: Interactive Behavior** in both command files (`prompt.md`, `prompt.toml`) — Response templates, error recovery table, smart defaults, formatting rules, and max question interactions.

### Changed

- **`SKILL.md`** — Rewrote Section 2 (Smart Routing) to align with canonical sk-doc template: prose Primary Detection Signal, split `score_intents()`/`select_intents()` two-function pattern, `_task_text()` helper, `UNKNOWN_FALLBACK_CHECKLIST`, `ON_DEMAND_KEYWORDS`, `RESOURCE_BASES` covering both `references/` and `assets/`. Removed INTERACTIVE intent from router. Updated Section 5 references. Version bumped to 1.2.0.0.
- **`README.md`** — Removed deleted file references from structure tree and key files table. Removed Multi-Format Output subsection. Updated operating modes count.
- **`references/depth_framework.md`** — Removed Creative energy level throughout: routing, round config table, perspectives, cognitive rigor, technique usage, mode-specific scoring subsection, quick reference table, must-have rules.
- **`references/patterns_evaluation.md`** — Removed visual framework sections (VIBE, VIBE-MP, FRAME, MOTION deep dives), selection algorithm visual blocks, design directions subsection, CLEAR vs EVOKE vs VISUAL comparison, entire Visual Scoring section. Renumbered remaining sections.
- **`.opencode/skill/README.md`** — Updated sk-prompt-improver entry: version v1.0.0 → v1.2.0.0, corrected framework list to 7 text frameworks, added to folder signals table.
- **`.opencode/command/create/prompt.md`** — Fixed stale `format_guides.md` references. Replaced `interactive_mode.md` loading with inline Step 1a.
- **`.agents/commands/create/prompt.toml`** — Same fixes as `prompt.md`: stale references and inline Step 1a.
