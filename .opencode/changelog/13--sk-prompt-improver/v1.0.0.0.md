## Highlights

### New

- **Skill created** - Converted standalone Prompt Improver AI system (v0.200) into an OpenCode-native skill at `.opencode/skill/sk-prompt-improver/`.
- **SKILL.md orchestrator** - 2,334 words with smart routing across 6 intent categories (TEXT_ENHANCE, VISUAL_UI, IMAGE_GEN, VIDEO_GEN, FRAMEWORK, FORMAT).
- **7 reference files** - Progressive disclosure architecture with bundled knowledge base:
  - `depth_framework.md` - DEPTH thinking methodology (Discover, Engineer, Prototype, Test, Harmonize) with energy-level scaling
  - `patterns_evaluation.md` - 11 frameworks (RCAF, COSTAR, RACE, CIDI, TIDD-EC, CRISPE, CRAFT, VIBE, VIBE-MP, FRAME, MOTION) with CLEAR scoring
  - `interactive_mode.md` - State machine with 18 states, 12 commands, 7 conversation variants
  - `visual_mode.md` - VIBE/VIBE-MP framework with EVOKE scoring for UI design prompts
  - `image_mode.md` - FRAME framework (30 sub-categories) with VISUAL scoring (60-point) for 9 AI image generators
  - `video_mode.md` - MOTION framework with VISUAL scoring (70-point) for 10 AI video platforms
  - `format_guides.md` - Consolidated Markdown, JSON, YAML format specifications
- **Skill advisor integration** - 18 intent boosters, 21 phrase boosters, 0.95 confidence routing for prompt-related queries.
- **Triple scoring** - CLEAR (50-point text), EVOKE (visual UI), VISUAL (image/video) quality scoring systems.
- **All references aligned** to `skill_reference_template.md` structure with proper ANCHOR open/close pairs, numbered sections, and `## 1. OVERVIEW` with Core Principle + When to Use.

### Architecture Decisions

- **ADR-001: Progressive Reference Strategy** - SKILL.md as lean orchestrator with 7 reference files for deep content (source material ~250KB).
- **ADR-002: Adapt Rather Than Copy** - Source docs adapted for OpenCode skill architecture, removing standalone project assumptions.
- **ADR-003: Consolidated Format Guides** - Three separate format guides merged into single `format_guides.md`.
