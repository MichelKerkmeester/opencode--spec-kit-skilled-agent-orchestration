**Integrated CocoIndex semantic search across spec_kit and create command systems** -- Added `mcp__cocoindex_code__search` to `allowed-tools` in all spec_kit and create command files, enhanced parallel explorer agent prompts with CocoIndex-first guidance, added Code Search Routing protocol blocks, and updated all `.agents/commands/` TOML mirrors. Agents now prefer semantic search for concept-based discovery over Grep pattern guessing.

## 1. OVERVIEW

CocoIndex (`mcp__cocoindex_code__search`) was declared MANDATORY in CLAUDE.md but had zero mentions across the spec_kit command system (8 .md, 17 YAML) and create command system (7 .md, 12 YAML). This release adds smart CocoIndex integration with a consistent Code Search Routing protocol: CocoIndex first for concept/intent queries, Grep only for exact known tokens. All `.agents/commands/` TOML mirrors were also updated to stay in sync.

## 2. HIGHLIGHTS

### Category: spec_kit Commands (21 files)

- **Explorer agent prompts enhanced** -- All 8 parallel explorer agents (4 in plan, 4 in complete) now explicitly instruct: "Use CocoIndex first for concept-based discovery, Grep only for exact known tokens"
- **Code Search Routing block** -- Reusable routing table added to plan.md and complete.md showing when to use CocoIndex vs Grep vs Glob vs memory_search
- **Deep research agents upgraded** -- `tools:` and `mcp_servers:` arrays updated in all 4 deep-research YAML variants to include CocoIndex
- **Debug agent prompt enhanced** -- Debug sub-agent dispatch template now includes CocoIndex guidance for finding similar working patterns
- **`allowed-tools` frontmatter** -- Added `mcp__cocoindex_code__search` to 7 spec_kit .md files (plan, complete, implement, deep-research, debug, phase, resume)

### Category: create Commands (19 files)

- **`allowed-tools` frontmatter** -- Added `mcp__cocoindex_code__search` to all 7 create .md files (sk-skill, feature-catalog, testing-playbook, agent, changelog, folder_readme, prompt)
- **YAML `code_search_note`** -- Added CocoIndex discovery guidance to `request_analysis_framework` in all 12 create YAML assets (6 auto + 6 confirm)

### Category: TOML Mirrors (14 files)

- **`.agents/commands/` TOMLs synced** -- Updated `allowed-tools` in all 14 applicable TOML command mirrors (6 spec_kit + 7 create + 1 deep-research) to include `mcp__cocoindex_code__search`
