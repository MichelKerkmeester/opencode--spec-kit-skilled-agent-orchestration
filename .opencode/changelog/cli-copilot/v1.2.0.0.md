This release explains the real non-interactive way to set reasoning depth and removes older workaround guidance. It also narrows the model list to a smaller set of practical recommendations. The goal is to make the workflow easier to follow and easier to trust.

---

#### A direct setup path for reasoning depth

The earlier guidance depended on a workaround that was harder to automate cleanly. The docs now explain the direct configuration setting that controls reasoning depth without interactive setup. Automated runs are easier to configure and easier to repeat.

---

#### Simpler model guidance

A very long model catalog made it harder to decide what to use. The recommended model list was trimmed to a smaller set of practical choices. New users can pick a sensible starting point more quickly.
