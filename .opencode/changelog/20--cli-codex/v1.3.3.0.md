## [**1.3.3.0**] - 2026-03-28

**Spec folder pass-through rule for CLI delegation** -- Added ALWAYS rule #8 requiring the calling AI to pass its active spec folder to the delegated Codex agent in the prompt. If no spec folder exists, the calling AI must ask the user before delegating. Prevents delegated agents from halting at Gate 3 in `--full-auto` or non-interactive mode.

## Highlights

### Category: Rules

- **New ALWAYS rule #8: spec folder pass-through** -- The calling AI must include `Spec folder: <path> (pre-approved, skip Gate 3)` in the delegation prompt when it has an active spec folder. If the calling AI does not have a spec folder, it must ask the user before delegating, since the delegated agent cannot answer Gate 3 interactively.
