## Highlights

### Added

- **Reasoning Effort Configuration Section** — New dedicated section in `cli_reference.md` documenting all 6 valid values (`none`, `minimal`, `low`, `medium`, `high`, `xhigh`) sourced from `codex-rs/core/config.schema.json`
- **Config Override Flag Documentation** — New `-c` / `--config` flag section in `cli_reference.md` with usage examples for runtime config overrides
- **Reasoning Effort Level Table** — Added to SKILL.md Model Selection with per-level use case guidance
- **Explicit "No --reasoning-effort Flag" Callout** — Added note in SKILL.md clarifying the correct syntax (`-c model_reasoning_effort="<level>"`) to prevent the common `--reasoning-effort` misuse

### Changed

- **SKILL.md Flags Table** — Added `-c model_reasoning_effort="<level>"` row to Core Invocation Pattern flags
- **SKILL.md Model Table** — GPT-5.4 reasoning effort column now says "configurable via `-c model_reasoning_effort`" instead of just "configurable"
- **cli_reference.md Essential Flags** — Added `--config` / `-c` to the essential flags table
- **integration_patterns.md Strength Comparison** — Updated GPT-5.4 reasoning depth from "configurable effort" to "configurable via `-c model_reasoning_effort`"
- **prompt_templates.md Flag Reference** — Added `-c model_reasoning_effort` row with valid values note
- **Skill Version** — Bumped from 1.1.0 to 1.2.0
