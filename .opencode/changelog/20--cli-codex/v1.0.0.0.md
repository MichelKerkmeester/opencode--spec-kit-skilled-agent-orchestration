## Highlights

### New

- **Cross-AI Orchestration** — Claude Code as conductor, Codex CLI as executor for supplementary AI tasks (code review, web research, architecture analysis, code generation)
- **7-Intent Smart Routing** — GENERATION, REVIEW, RESEARCH, ARCHITECTURE, TEMPLATES, PATTERNS, AGENT_DELEGATION with canonical `score_intents()` / `select_intents()` pattern
- **Codex Agent Delegation** — Routing table for 9 specialized Codex agents (context, debug, handover, orchestrate, research, review, speckit, ultra-think, write) via `.codex/agents/*.toml`
- **Sandbox-Controlled Execution** — Three-tier sandbox model (`read-only`, `workspace-write`, `danger-full-access`) with safety rules per task type
- **Single Model Standardization** — `gpt-5.3-codex` with `xhigh` reasoning effort as the only supported model
- **Skill Advisor Integration** — INTENT_BOOSTERS, MULTI_SKILL_BOOSTERS, PHRASE_INTENT_BOOSTERS entries in `skill_advisor.py` (0.95 confidence on "use codex" queries)
- **Documentation (7 files)** — SKILL.md (8 sk-doc sections), README.md, `references/cli_reference.md`, `references/integration_patterns.md`, `references/codex_tools.md`, `references/agent_delegation.md`, `assets/prompt_templates.md`
- **Unique Capabilities** — `/review` command (diff-aware code review), `--search` (live web browsing), `--image` (visual input), session management (resume/fork), MCP server integration, cloud execution
