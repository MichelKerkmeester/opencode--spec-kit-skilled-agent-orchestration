## [**1.0.0**] - 2026-03-08

This first release adds a ClickUp skill that can switch between a quick command-line path and a fuller tool-integration path. It is designed to cover both daily task work and more advanced project-management workflows.

#### Choose the right path for the job

- Need: Quick task work and advanced project workflows did not need the same level of tooling.
- Change: The new skill routes between a lightweight path for everyday work and a fuller path for more complex operations.
- Why it matters: Daily work stays simple while advanced capabilities are still available when needed.

&nbsp;

#### Complete support package

- Need: New users needed more than a short overview to adopt the skill confidently.
- Change: The release shipped a main guide, install guide, command reference, workflow examples, and a categorized tool map.
- Why it matters: People can onboard, compare options, and troubleshoot from one package.

&nbsp;

#### Better routing into the new skill

- Need: ClickUp requests were too easy to send to a broader tool instead of this focused skill.
- Change: The routing rules and the skills catalog were updated so ClickUp work points here more directly.
- Why it matters: Users are more likely to land in the right tool on the first try.
