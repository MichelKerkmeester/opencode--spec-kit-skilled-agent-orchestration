This release adds a feature catalog for `sk-improve-agent` so you can see what the evaluation stack actually does without reading source code. The catalog lists 13 features grouped into 3 areas and points to the code file for each one.

> Spec folder: `.opencode/specs/system-spec-kit/026-graph-and-context-optimization/011-skill-advisor-graph/008-deep-skill-feature-catalogs/` (Level 3)

---

## What Changed

#### New feature catalog

Before this release, the only way to understand what `sk-improve-agent` did was to read `SKILL.md` or the source files. No map of the evaluation loop, no breakdown of the integration scanner, no pointer from a scoring rule back to the code that enforces it.

The new catalog at `.opencode/skill/sk-improve-agent/feature_catalog/feature_catalog.md` groups the skill into 3 areas:

- Evaluation loop: setup, candidate generation, integration-aware evaluation, promotion gate, rollback path
- Integration scanning: cross-surface dependency discovery, contract scan, runtime-signal capture
- Scoring system: 5-dimension rubric, score aggregation, margin computation, threshold gates, deterministic replay

Each feature has its own file with an overview, what the code actually does today and a pointer to the source.

#### README link

The skill README now points to the catalog as the first place to look when you want to understand the evaluation stack.
