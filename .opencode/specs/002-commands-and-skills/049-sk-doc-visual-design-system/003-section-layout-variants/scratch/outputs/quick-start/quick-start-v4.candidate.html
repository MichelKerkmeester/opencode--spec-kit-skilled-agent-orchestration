Quick Start

Recipe Sequence

Execute these commands sequentially to stand up your local environment. Total estimated time: ~3 minutes.

00

Prerequisites

  • Template CLI v0.0.0+ opencode --version
  • Node.js 18.x or higher node --version
01

Acquire & Install

Clone the repository locally and install the core dependencies required for the build.

# Clone repository
git clone https://github.com/example/repo.git
cd repo

# Install dependencies
npm install
02

Initialize Services

Start the local development server to verify the installation was successful.

$ 
npm run dev
Server will be available at http://localhost:3000
