Dependency Gate Cards

Each card acts as an explicit entry condition. Teams can validate readiness in any order while keeping command intent and verification output tightly paired.

CLI Availability

Required

Confirm local command surface is installed.

opencode --version
Expect semantic version output

Runtime Version

Required

Ensure Node baseline is compatible with tasks.

node --version
Expect v18.x or higher

Package Resolution

Blocking

Resolve dependencies before start commands.

npm install
Expect install summary without errors

Service Boot

Blocking

Launch dev service and verify endpoint.

npm run dev
Expect localhost endpoint announcement
