Tier 1 & 2: Process Playbooks

{{issue1_name|default("Service Degradation / Timeout")}}

Playbook ID: TS-01-NET

Severity: Medium
Symptoms & Triggers
  • API requests drop by > 15%
  • Latency spikes above 500ms
  • Client reports of 504 Gateway Timeouts
Action Plan

  1. Verify Load Balancer

    Check active node count in the infrastructure dashboard.

  2. Restart Stalled Workers

    pm2 reload all

Escalation Checkpoint

If latency does not recover within 5 minutes, escalate to Tier 3.

{{issue2_name|default("Build Pipeline Failure")}}

Playbook ID: TS-02-CI

Severity: Low
Symptoms & Triggers
  • Automated tests fail on main branch
  • Docker image build step errors out
Action Plan

  1. Clear Build Cache

    Navigate to CI settings and purge cache directory.

  2. Re-trigger Pipeline

    Run pipeline manually with empty cache parameter.

Standard Resolution

If failure persists, revert last commit. No page required.
