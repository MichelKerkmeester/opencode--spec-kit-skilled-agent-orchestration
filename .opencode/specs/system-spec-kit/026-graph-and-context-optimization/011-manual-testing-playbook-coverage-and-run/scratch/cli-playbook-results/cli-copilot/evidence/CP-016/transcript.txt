**Yes — MCP is supported natively.** The Copilot CLI docs say it has **“MCP-powered extensibility”**, that it **“ships with GitHub’s MCP server by default and supports custom MCP servers,”** and that you manage them with **`/mcp`**. The official “Using GitHub Copilot CLI” doc is more specific: **“Copilot CLI comes with the GitHub MCP server already configured”** and you can add another one with **`/mcp add`**; server definitions are stored in **`~/.copilot/mcp-config.json`** by default, or under a different base dir if you set **`COPILOT_HOME`**. The help text also documents **`/env`** for showing loaded environment details, including **MCP servers**.
