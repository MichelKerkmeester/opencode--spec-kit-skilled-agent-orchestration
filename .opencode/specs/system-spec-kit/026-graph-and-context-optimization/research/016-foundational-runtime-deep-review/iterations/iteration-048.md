## Novel Insights
- The open Gate 3 problem is not only false positives from words like `phase`; the same prose detector also misses the repository's own canonical save command surface. String governance is now failing on both sides of the classifier boundary.
- The YAML command assets have a deeper schema problem than "booleans are strings": the control key itself is overloaded. `when:` no longer means one thing, so even a future validator cannot enforce a single grammar without first reverse-engineering intent from prose.
- The most striking cross-system pattern in this pass is asymmetry of rigor: memory-save phrases are mechanically recognized in routing/tests, but the pre-flight safety gate that is supposed to run earlier still relies on a narrower English list.
