**Q1 Synthesis**

The best pipeline improvement is to make hybrid Stage 1 truly candidate-only: split `hybrid-search.ts` into a raw channel collector/fuser that Stage 1 can call, then keep Stage 2 as the only signal-scoring layer and Stage 3 as the only rerank/MMR/MPAB layer. In parallel, tighten Stage 3 so fallback reranker output is treated as “not applied,” and merge deep/R12 variants by max/effective score or by a second fusion pass instead of insertion order.
