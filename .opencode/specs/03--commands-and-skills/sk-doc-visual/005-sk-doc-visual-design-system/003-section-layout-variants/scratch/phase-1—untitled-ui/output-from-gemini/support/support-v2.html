How do I install the CLI tool?

The fastest way to install the tool is using npm or Homebrew, depending on your environment. For most web development setups, we recommend the global npm package.

npm install -g @untitled/cli
untitled login

Ensure you have Node.js version v18.0.0 or higher installed.

Yes, we offer automated migration scripts for major platforms. You can export your data as a JSON file and run our migration command. Please refer to our Migration Guide for step-by-step instructions.

What are the system requirements?

Our tools are lightweight and run on Windows, macOS, and Linux. Minimum requirements are 2GB RAM and 500MB of free disk space. A stable internet connection is required for initial authentication.

Configuration & Troubleshooting

2 questions
Why am I getting a 'Connection Refused' error?

This typically happens when your firewall blocks the outgoing connection to our authentication servers, or if you are behind a corporate VPN that requires custom proxy settings.

You can configure proxy settings by exporting the HTTP_PROXY environment variable before running the application.

How do I reset my configuration to default?

You can safely delete the local configuration file located at ~/.untitled/config.json. The application will generate a fresh default configuration on the next startup.
