Developer Guide

Build Your First Extension

Follow this 6-step lifecycle to scaffold, develop, and publish a custom extension for the platform.

1. Global CLI Installation

First, ensure you have the Untitled CLI tools installed globally on your machine. This requires Node.js v18 or higher.

npm install -g @untitled/cli
2

2. Scaffold the Project

Use the CLI to generate a new extension boilerplate. The interactive prompt will ask for your extension name, author details, and preferred template (TypeScript/JavaScript).

untitled ext init my-awesome-plugin
cd my-awesome-plugin
npm install
3

3. Develop & Implement Hooks

Open src/index.ts and start implementing the required lifecycle hooks. Use the local development server to test changes with hot-reloading.

untitled dev --watch
4

4. Package for Production

Once development is complete, run the build command. This compiles your TypeScript, bundles dependencies, and generates the final dist/plugin.zip artifact.

npm run build
5

5. Publish to Marketplace

Authenticate with your developer account and push the extension to the registry. It will undergo automated security scanning before being listed.

untitled ext publish --token=$DEV_TOKEN
