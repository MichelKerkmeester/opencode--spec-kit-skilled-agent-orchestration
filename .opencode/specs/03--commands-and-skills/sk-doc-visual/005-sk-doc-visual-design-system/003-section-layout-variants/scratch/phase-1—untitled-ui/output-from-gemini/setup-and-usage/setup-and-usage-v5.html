Install Package

Add the core library to your existing React or Next.js project. We support npm, yarn, and pnpm.

npm install @untitled-ui/core
2

Initialize Configuration

Run the CLI initialization command to generate a default configuration file at the root of your project.

This will automatically detect if you are using Tailwind and adapt the generated configuration accordingly.

npx untitled init
3

Wrap with Provider

Add the ThemeProvider to your root layout file to ensure all components receive the correct context and tokens.

import { ThemeProvider } from '@untitled-ui/core'; export default function App({ children }) { return <ThemeProvider>{children}</ThemeProvider>; }
