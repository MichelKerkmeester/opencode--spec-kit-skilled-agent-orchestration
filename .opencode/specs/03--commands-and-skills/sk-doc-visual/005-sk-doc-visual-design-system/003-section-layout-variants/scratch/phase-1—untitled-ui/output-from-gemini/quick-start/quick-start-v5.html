Installation

Follow these steps to add UntitledUI to an existing React or Next.js project and prepare your environment for development.

Step 1

Add dependencies

First, install the core library and its peer dependencies via npm, yarn, or pnpm. We recommend using @untitled/ui-react for standard web applications.

npm install @untitled/ui-react framer-motion lucide-react
> added 42 packages, and audited 43 packages in 2s
> 12 packages are looking for funding
> found 0 vulnerabilities
Step 2

Run the init CLI

The CLI tool will automatically detect your framework (e.g., Next.js, Vite) and scaffold the necessary global CSS and configuration files in your root directory.

npx @untitled/cli init
✓ Detected framework: Next.js (App Router)
✓ Creating theme.config.ts...
✓ Injecting CSS variables into app/globals.css...
✓ Initialization complete!
