Interactive Quick Start

Follow these four steps to scaffold your new application, configure the local environment, and start the development server.

Watch Video Tutorial Read Full Docs

Initialize Project

Scaffold a new project using our interactive CLI. This creates the required folder structure and base files.

$ npx create-untitled@latest
Advanced CLI flags
Use --template typescript to enforce strong typing, or --skip-install to bypass the initial dependency resolution.
2

Configure Variables

Copy the sample environment file to securely establish local credentials and API keys without committing them.

$ cp .env.example .env
Required Tokens
You must supply a valid NEXT_PUBLIC_API_URL to connect to the staging database before proceeding to dependency installation.
3

Install Dependencies

Resolve and download all required node modules defined in your package.json lockfile.

$ npm install
Peer Dependency Issues?
If you encounter React version mismatches on legacy systems, try appending the --legacy-peer-deps flag to your install command.
4

Start Development

Boot up the local hot-reloading server. Your project will automatically become available in your default browser.

$ npm run dev
Custom Port Override
By default, the server binds to localhost:3000. To use a different port, execute npm run dev -- -p 4000 instead.
