          <h2 id="authentication-flow">Standard Flow</h2>
          <p>All API requests must include a valid Bearer token in the Authorization header. Obtain this token by exchanging your application credentials at the token endpoint.</p>

          <pre class="code-block"><code>curl -X POST https://api.untitled.com/v2/oauth/token \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&client_id=YOUR_ID&client_secret=YOUR_SECRET"</code></pre>

          <h2 id="scopes">Scopes and Permissions</h2>
          <p>Tokens can be restricted via scopes. Standard scopes include:</p>
          <ul>
            <li><span class="code-inline">read:profile</span> — Access basic user profile data.</li>
            <li><span class="code-inline">write:profile</span> — Modify user profile data.</li>
            <li><span class="code-inline">admin:users</span> — Full administrative access to tenant user pool.</li>
          </ul>
