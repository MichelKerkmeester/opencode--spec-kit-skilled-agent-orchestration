Setup & Installation

Follow these steps to get your environment configured and ready for local development. We recommend doing this sequentially.

1

Prerequisites

Verify system requirements

Before installing the package, ensure your local development environment meets the following minimum requirements:

  • Node.js v18.0.0 or higher
  • npm v9.0.0 or higher (or pnpm v8)
  • Git installed and configured

You can check your current Node version by running:

node --version
npm --version
2

Install Dependencies

Clone the repo and install packages

Clone the repository to your local machine and navigate into the project directory. Then, install the required dependencies using your preferred package manager.

git clone https://github.com/untitled-ui/framework.git
cd framework
npm install

Dependencies installed successfully

A node_modules folder and package-lock.json should now be present in your directory.

3

Environment Configuration

Set up your local .env file

Copy the example environment file to create your local configuration. You will need to obtain API keys from your dashboard to complete the setup.

cp .env.example .env

Open .env in your text editor and update the following required values:

NEXT_PUBLIC_API_URL=http://localhost:3000/api
DATABASE_URL=postgresql://user:password@localhost:5432/db
AUTH_SECRET=generate_a_random_32_char_string
4

Database Migrations

Initialize the database schema

With your DATABASE_URL configured, run the following command to push the schema to your local database instance and generate the client.

npx prisma db push
npx prisma generate

Optionally, you can seed the database with test data by running npm run db:seed.

5

Start Development Server

Run the app locally

Final Step

Start the local development server. It will watch for file changes and hot-reload the application automatically.

npm run dev

Server is running

Open http://localhost:3000 in your browser to see the application.
