Installation Guide

Learn how to install and configure UntitledUI in your project. We support a variety of package managers and frameworks to get you up and running in minutes.

or use npm/yarn 
# Install via npm
npm install @untitled-ui/core

# Or via yarn
yarn add @untitled-ui/core

// Then import in your root file
import { ThemeProvider } from '@untitled-ui/core';
import '@untitled-ui/core/styles.css';

setupUntitled({
  theme: 'light',
  cssVariables: true
});
