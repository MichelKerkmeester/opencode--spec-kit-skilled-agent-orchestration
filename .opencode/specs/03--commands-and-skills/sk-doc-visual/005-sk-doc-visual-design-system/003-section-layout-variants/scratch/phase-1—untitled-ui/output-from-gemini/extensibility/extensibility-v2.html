Retrieve a project

Retrieves the details of an existing project. You need only supply the unique project identifier that was returned upon project creation.

GET https://api.untitled.com/v2/projects/{id}

Path Parameters

Parameter Description
idRequired string

The unique identifier of the project to retrieve. Must be a valid UUID v4.

Query Parameters

Parameter Description
expand array of strings

Specifies which relation fields should be expanded in the response. Available fields are owner, team, and active_sprint.
include_archived boolean

If set to true, the response will include child resources (like tasks) even if they are marked as archived. Default is false.

Response

Returns a project object if a valid identifier was provided. Returns a 404 error if the project does not exist or you do not have permission to access it.
