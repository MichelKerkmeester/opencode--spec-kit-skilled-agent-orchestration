Critical ERR_DB_TIMEOUT
Connection pool exhausted during peak load, or primary database replica is unreachable due to network partition.

Resolution Protocol

  1. Verify primary database replica health via the monitoring dashboard.
  2. If the database is healthy, check the connection pool limits in the API configuration.
  3. Restart the service pods to cleanly flush existing hanging connections.
kubectl rollout restart deployment/api-service -n production
kubectl logs -f deployment/api-service -n production

Warning AUTH_TOKEN_EXPIRED
User session token has naturally expired, but the client failed to automatically perform a silent token refresh.

Resolution Protocol

  1. Instruct the user to manually clear their browser cache and cookies for the domain.
  2. Verify the frontend client is properly catching 401 Unauthorized responses.
  3. Ensure the refresh token rotation is enabled in your identity provider settings.

Info SYNC_DELAY_DETECTED
Background job queue is processing tasks slower than normal due to large batch imports from third-party integrations.

Resolution Protocol

  1. No immediate action required unless the delay exceeds 15 minutes.
  2. Scale the background worker nodes if the queue depth continues to rise linearly.

Critical CORS_ORIGIN_DENIED
An unauthorized client application attempted to access the API, or a newly deployed frontend domain is missing from the allowed origins list.

Resolution Protocol

  1. Check the Origin header in the blocked request to identify the client domain.
  2. Update the API configuration to include the missing domain if it is legitimate.
  3. Ensure the changes are deployed to the gateway proxy.
