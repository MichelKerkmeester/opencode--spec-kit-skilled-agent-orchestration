[`vector-index-mutations.ts#L410`](/Users/michelkerkmeester/MEGA/Development/Opencode%20Env/Public/.opencode/skill/system-spec-kit/mcp_server/lib/search/vector-index-mutations.ts#L410) sets success metadata before the vector write path at [`vector-index-mutations.ts#L428`](/Users/michelkerkmeester/MEGA/Development/Opencode%20Env/Public/.opencode/skill/system-spec-kit/mcp_server/lib/search/vector-index-mutations.ts#L428). That is a real corruption path for vector retrieval.
