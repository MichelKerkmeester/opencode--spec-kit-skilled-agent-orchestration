4. **No CLI-level hooks exist -- confirmed.** Claude Code's `.claude/settings.local.json` contains only MCP tool permissions (allowedTools). [INFERENCE: based on template-structure.js contract scope (headers + anchors only) vs. Level 3+ template content]
